    🛡️ Segurança & Minimização de Exposição

    🎯 O MascaraAI foi criado para resolver uma dor real: prontuários e dados clínicos sensíveis que precisam ser compartilhados com IAs externas sem expor o paciente. Nossa abordagem foca na minimização máxima da superfície de exposição de PII/PHI antes de qualquer transmissão — reduzindo drasticamente o risco de vazamento de dados em fluxos clínicos automatizados.

    Arquitetura Zero-Data-Residue

    O Mascara AI foi projetado sob o princípio de Privacy by Design. Quando você utiliza o Playground local, seus dados clínicos reais nunca saem da sua máquina. O processamento de inteligência artificial (NER) e as expressões regulares rodam 100% na memória RAM local do seu navegador através de um Web Worker isolado.

    Trânsito Criptografado & Anonimização de LLMs

    Ao utilizar a integração de IA para processar prontuários, o texto original é higienizado e substituído por tags (ex: [NOME:1]) localmente antes de trafegar na rede. Apenas o texto limpo é enviado para as APIs externas da OpenAI/Groq/Gemini. A restauração ocorre novamente no seu próprio navegador, reduzindo significativamente a exposição de dados pessoais durante o processamento.

    Alinhamento aos Princípios da LGPD (Lei 13.709/18)

    O MascaraAI foi projetado para apoiar estratégias de minimização, pseudonimização e governança de dados pessoais de saúde, alinhadas aos princípios dos Artigos 13 e 14 da LGPD brasileira, como apoio ao tratamento de dados de saúde para fins de apoio à decisão clínica, sem substituir as demais obrigações legais e o dever de sigilo profissional do médico (Código de Ética Médica).

    Licença e Tecnologias de Código Aberto

    Para garantir transparência e auditabilidade total exigidas na área da saúde, o MascaraAI é um software de código aberto (licenciado sob AGPLv3). O motor de Processamento de Linguagem Natural (NLP) utiliza como base o modelo de linguagem BERTimbau (desenvolvido por NeuralMind) e o dataset leNER-Br, ambos distribuídos sob a Licença Apache 2.0.

    Termos de Uso e Privacidade

    O MascaraAI é focado na minimização de exposição de dados clínicos sensíveis antes que estes sejam transmitidos para modelos de inteligência artificial de terceiros.

    1. Plano Free Browser (Processamento Local): Todo o processamento ocorre localmente na memória RAM do seu navegador. Nenhum dado clínico é transmitido pela rede, minimizando o risco de vazamento.

    2. Integrações e APIs (Processamento Ativo no seu Servidor): Com exceção do processamento manual no navegador, todas as integrações de API (tanto Free API quanto Plano Profissional) processam os dados enviando-os para o seu próprio servidor ativo rodando o motor híbrido (Regex + NER). Isso garante total soberania de dados sob a gestão do próprio usuário.

    ⚠️ Validação de CPF e CNPJ: Para evitar falsos positivos, nosso motor de privacidade utiliza validadores matemáticos de dígitos verificadores. Sequências numéricas que imitem o formato de documentos mas que não sejam matematicamente válidas não serão tratadas ou mascaradas pelo sistema como CPF/CNPJ, cabendo ao usuário auditar a validade das informações.

    Auditoria e Isenção de Responsabilidade Final: O MascaraAI é uma ferramenta de auxílio na minimização de dados e mitigação de riscos de exposição de prontuários. Como todo sistema de IA e heurísticas, ele pode cometer erros de classificação. A conferência final do texto descaracterizado e a responsabilidade exclusiva pelo compartilhamento de informações permanecem integralmente com o usuário.
